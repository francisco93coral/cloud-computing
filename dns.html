DNS
Un Servidor DNS (Domain Name System) se utiliza para proveer a las computadoras de los usuarios (cliente) un nombre equivalente a las direcciones IP. El uso de este servidor es transparente para los usuarios cuando éste está bien configurado.
Cada LAN (Local Área Network) debería contar con un servidor DNS. Estos servidores trabajan para intercambiar información y obtener las direcciones IP de otras LANs.

BIND
BIND (Acronimo de Berkely Internet Name Domain) es una implementación del protocolo DNS y provee una implementación libre de los principales componentes del sistema de nombres de dominio, las cuales incluyen:

  • Un servidor de sistema de nombres de dominio (named).
  • Una Biblioteca resolutoria de sistema de nombres de dominio.
  • Herramientas para verificar la operación adecuada del servidor DNS (bind-utils).
