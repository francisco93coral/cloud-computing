Servicio FTP

FTP (siglas en inglés de File Transfer Protocol, ‘Protocolo de Transferencia de Archivos’), es un protocolo de red para la transferencia de archivos entre sistemas interconectados o enlazados a Internet, basado en la arquitectura cliente-servidor. Desde un equipo cliente se puede conectar a un servidor para descargar archivos desde él o para enviarle archivos, independientemente del sistema operativo utilizado en cada equipo.

Manual de Creación del Servicio FTP Ubuntu

  1. Ingresa al terminal y digitar: sudo su

    A continuación se digita el pasword

  2. En el terminar se ingresa el comando: apt-get install vsftpd
  3. A continuación creamos un grupo de usuarios ftp con el comando: groupadd ftp
  4. Se crea la carpeta para el uso del ftp con el comando: mkdir /home/ftp
  5. Se crea la carpeta para el usuario del ftp con el comando: mkdir /home/ftp/ubuntucliente
  6. Se crea la carpeta para el bin del ftp con el comando: mkdir /bin/ftp
  7. A continuación modifica el shell para seguidad del sistema operativo con el comando: nano /etc/shells

    8. Al final del archivo se agrega /bin/ftp

  8. Se crea un usuario para el acceso al servicio ftp con el comando:
    useradd -g ftp -d /home/ftp/ubuntucliente -s /bin/ftp ubuntucliente
  9. Se agrega el pasword para el cliente del servicio ftp con el comando: passwd ubuntucliente
  10. Ahora hay que darle permiso al usuario pueda ser el dueño de su carpeta con los comandos:

    cd /home/ftp

    chown ubuntucliente.ftp ubuntucliente/
  11. Abrimos el archivo de configuración con el comando: nano /etc/vsftpd.conf

    Borra el segno de # en las siguientes filas:

    anonymous_enable=NO
    local_enable=YES
    write_enable=YES
    local_umask=022
    chroot_local_user=YES
    chroot_list_enable=YES
    chroot_list_file=/etc/vsftpd.chroot_list
  12. Finalmente agregamos en la lista al usuario el comando:
    sudo touch vsftpd.chroot_list
    echo “usuario”>> vsftpd.chroot_list
  13. Finalmente reiniciamos el servicio con el comando: service vsftpd restart
